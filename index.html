ICS-213 Form

Introduction

This web page was designed for receiving messages via radio. Type in the message into the appropriate fields. Use the tab key to move between fields.

The form is designed to print to a single page. Use the browser's print preview function to verify. Adjust the scaling percentage or margins if needed.

If something goes wrong, hard-refresh the page. On Windows and Linux, ctrl-shift-R. On Mac, cmd-shift-R.

Saving Form Data

Optionally, save the form data to a file using the "Save Form Data" button at the bottom of the page. Files are typically saved in the Downloads folder, but the folder can be changed through browser settings. The files have a .json extension.

Reloading Form Data

The button at the top of the form, labeled "Browse…", will reload saved form data.

Resetting the Form

It is best to reset the form before working on the next message. Use the browser's hard refresh function, e.g., shift-ctrl-R or shift-F5.

Cookies

When you the save the form data, the form will remember the Operator Call Sign and Operator Name. The operator fields will be pre-filled when you refresh the browser.

The "Clear Cookies" button will make the form forget the call sign and name.
